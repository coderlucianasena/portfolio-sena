# Minificação e Concatenation:
- Minificação: Remove caracteres desnecessários do código, como espaços, quebras de linha e comentários, sem afetar sua funcionalidade.
- Concatenation: Junta vários arquivos CSS ou JavaScript em um único arquivo para reduzir o número de requisições HTTP, melhorando o tempo de carregamento.

## Ferramenta para minificação e concatenation
- Gulp:
Visão Geral: Gulp é uma ferramenta muito simples e poderosa para automação de tarefas. Ele é altamente personalizável e é ótimo para projetos que não precisam de toda a complexidade do Webpack. Gulp é ideal para pequenas tarefas como minificação, concatenation, e pré-processamento de CSS (como SASS ou LESS).
Quando usar: Para um projeto de portfólio ou pequenos sites, Gulp é uma excelente escolha por ser fácil de configurar e focado em tarefas simples.
